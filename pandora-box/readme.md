# Pandora Box - Syndicate Vault

Diretório de armazenamento de evidências, hipóteses e descobertas da investigação.

## Estrutura
- `evidencias/` - Análises forenses e documentos
- `hipoteses/` - Teorias e deduções estratégicas  
- `perfis/` - Análises psicológicas de personagens
- `timeline/` - Linha do tempo de eventos
